	How to use.

	./RTv1 {file_name}

	file should have for example
		-shapes and theyr properties and position
		-camera position
		-light and its properties and position




	Examples for everything you can put to the file.

	As meny as you want in any combination.
	{
		sphere 0.0.10   2   0xffffff00
		       pos.     r.  color

		plane 10.0.10   1.0.0   0xffffff00
		      pos       dir     color

		cylinder 0.0.10   10.0.1000    2  0xffffff00
		         srt.pos  heding.pos   r. color

		cone  0.0.10  0.10.10   2  0xffffff00
		      srt.pos top.pos   r. color
	}

	Onely one per file.
	{
		light 0.0.0
		      pos

		camera 0.0.0 0.0.10             0.10.0
		       pos.  point of intres.    world up.
	}

	to Exit press ( esc )
	to reload image ( r )
